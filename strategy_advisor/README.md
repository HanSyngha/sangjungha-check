# 삼책지계 (三策之計)

## 앱 소개

**삼책지계**는 삼국지의 전설적인 책사들에게 인생의 고민을 상담받는 앱입니다.

현대인의 복잡한 고민을 삼국지 시대의 지혜로 풀어드립니다. 당신의 이야기를 들려주면, 책사가 **상책(上策)**, **중책(中策)**, **하책(下策)** 세 가지 전략을 제시합니다.

## 책사 소개

### 제갈량 (諸葛亮) - 와룡선생
> *"천하를 내다보는 지략가"*

신중하고 깊은 통찰력으로 장기적인 관점에서 최선의 전략을 제시합니다. 인의(仁義)를 중시하며 정도(正道)의 해결책을 찾아드립니다.

### 방통 (龐統) - 봉추선생
> *"역발상의 천재 전략가"*

대담하고 파격적인 계책으로 불가능을 가능으로 만듭니다. 연환계를 고안한 창의적 사고로 기존의 틀을 깨는 해결책을 제시합니다.

### 초선 (貂蟬) - 경국지색
> *"심리전의 대가"*

사람의 마음을 꿰뚫어 보는 통찰력으로 인간관계와 감정의 흐름을 읽습니다. 부드러움 속에 숨은 강인함으로 관계 속 해법을 찾아드립니다.

## 상책, 중책, 하책

- **상책 (上策)**: 가장 이상적이고 근본적인 해결책. 시간과 노력이 들지만 최선의 결과를 가져옵니다.
- **중책 (中策)**: 현실적인 타협안. 실행 가능하고 균형 잡힌 방법입니다.
- **하책 (下策)**: 급할 때의 임시방편. 빠르지만 부작용이 있을 수 있습니다.

## 사용 방법

1. 상담받고 싶은 책사를 선택합니다
2. 당신의 고민을 자유롭게 적어주세요
3. 책사가 추가 질문을 할 수도 있습니다
4. 세 가지 책략 중 당신에게 맞는 길을 선택하세요
