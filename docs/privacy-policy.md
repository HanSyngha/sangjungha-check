# 개인정보 처리방침

**삼지묘책** (이하 "앱")은 사용자의 개인정보를 중요하게 생각합니다.

## 1. 수집하는 개인정보

본 앱은 **개인정보를 수집하지 않습니다.**

- 회원가입 없음
- 로그인 없음
- 사용자 입력 내용은 기기에만 저장되며 서버로 전송되지 않음

## 2. 제3자 서비스

### Google AdMob (광고)
본 앱은 Google AdMob을 통해 광고를 표시합니다. AdMob은 광고 제공을 위해 다음 정보를 수집할 수 있습니다:
- 기기 식별자
- IP 주소
- 광고 상호작용 데이터

자세한 내용은 [Google 개인정보처리방침](https://policies.google.com/privacy)을 참조하세요.

### xAI API
사용자가 입력한 고민 내용은 AI 응답 생성을 위해 xAI API로 전송됩니다. 이 데이터는 응답 생성 후 저장되지 않습니다.

## 3. 데이터 보관

- 상담 기록은 사용자 기기에만 로컬 저장됩니다
- 앱 삭제 시 모든 데이터가 삭제됩니다
- 서버에 사용자 데이터를 저장하지 않습니다

## 4. 아동 개인정보

본 앱은 만 13세 미만 아동의 개인정보를 의도적으로 수집하지 않습니다.

## 5. 개인정보 처리방침 변경

본 방침이 변경될 경우 앱 업데이트를 통해 공지합니다.
